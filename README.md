# daylog

    How much time did you waste today?

### 오늘 얼마나 시간을 낭비했나요? 

하루의 계획을 기록하는 것이 아닌, 하루의 지난 일과를 기록하는 데일리리포트 어플리케이션 입니다. 현재 포트폴리오용으로 개발중이며 앱, 웹으로 서비스 예정입니다.

## 사용기술

- React
- Styled Component
- Apollo (Apollo Hooks)
- Node JS
- GraphQL
- Prisma
- nodemailer + sendGrid
- sharp + multer + AWS s3
- axios
